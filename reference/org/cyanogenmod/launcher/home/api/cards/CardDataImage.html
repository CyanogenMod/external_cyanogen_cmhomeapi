Class Overview

Class representing an image that should be displayed along with a parent CardData. Any number of CardDataImage objects can be associated with a CardData, and as many as possible will attempt to be displayed with the Card in CM Home. As with CardData, a CardDataImage will be published when publish(Context) is called. CardDataImage objects can similarly be removed from view by calling unpublish(android.content.Context).
